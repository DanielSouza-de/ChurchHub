## 📖 Sobre o projeto

O ChurchHub é um sistema web Full Stack desenvolvido para facilitar o gerenciamento de igrejas através de uma plataforma moderna e intuitiva.

A aplicação é composta por um site público para visitantes e um painel administrativo protegido por autenticação JWT, permitindo o gerenciamento completo dos conteúdos exibidos no site.

O projeto foi desenvolvido com foco em arquitetura em camadas, boas práticas de desenvolvimento e utilização do ecossistema Spring Boot para construção de APIs REST.

---

## ✨ Funcionalidades

### 🌐 Site Público

- Página inicial dinâmica
- Listagem de encontros
- Listagem de ministérios
- Informações da igreja carregadas pela API
- Links para redes sociais e contatos
- Layout responsivo

### 🔒 Painel Administrativo

- Login com autenticação JWT
- Dashboard administrativo
- Cadastro de encontros
- Edição de encontros
- Exclusão de encontros
- Cadastro de ministérios
- Edição de ministérios
- Exclusão de ministérios
- Alteração das configurações do site
- Logout seguro
- Toasts de sucesso e erro
- Modais de confirmação

### ⚙️ Backend

- API REST
- Spring Security
- JWT
- Spring Data JPA
- Bean Validation
- Swagger/OpenAPI
- Tratamento global de exceções
- CORS
- Arquitetura em camadas
